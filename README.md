Random Video Game Quote Generator
By Brendan
Course COSC-1436

This is a C++ application that I made for my final project in computer science class. This reads a database of over 50 videogame quotes from very well-known videogames such as: Halo Reach, Skyrim, Read Dead Redemption and a whole lot more to be able to interact with them through a random generator via menu

Features include:
* *Random Quote*: Uses a mathematical randomizer to pick a quote from the list.
* *Show All:* Displays every quote in the system with a numbered list.
* *Clear Screen:*Clears the console window for a better user experience.
* *File I/O:* Loads data dynamically from a text file into a Vector.

How to Run the program:
1. Open the project in Visual Studio
2. Ensure vquotes.txt is in the same directory as main.cpp
3. Build and Run the project.
